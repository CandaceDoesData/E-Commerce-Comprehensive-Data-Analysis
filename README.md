# E-Commerce-Comprehensive-Data-Analysis
## Overview
This project analyzes the Brazilian E-Commerce Public Dataset by Olist to uncover insights into customer
behavior, product performance, sales trends, and delivery operations. The goal was to build an end-to-end
analytics solution that demonstrates the complete data analytics workflow—from raw CSV files to an
interactive executive dashboard.
         
Using PostgreSQL, SQL, Python, and Tableau, I designed a relational database, transformed and validated
the data, performed exploratory and advanced SQL analysis, and created interactive visualizations to
support business decision-making.

## Business Problem
Retail organizations generate large volumes of transactional data, but raw data alone does not provide
actionable insights. This project demonstrates how data can be transformed into meaningful business
intelligence that helps answer questions such as:
  - Which products and categories generate the most revenue?
  - How do sales trends change over time?
  - Which regions have the highest customer demand?
  - How does delivery performance impact customer satisfaction?
  - What opportunities exist to improve operational performance?

## Objectives
1. Design a relational database from multiple CSV files
2. Perform data cleaning and validation
3. Analyze customer purchasing behavior and sales performance using SQL
4. Create reusable database views for reporting
5. Build an interactive Tableau dashboard for executive reporting
6. Present findings through data storytelling and business recommendations

## Project Workflow
This project follows an end-to-end analytics pipeline, beginning with raw data and ending with an
interactive business intelligence dashboard.

    Kaggle Olist Dataset (CSV Files)
    │
    ▼
    Python (Jupyter Notebook)
    • Import and inspect raw data
    • Validate data types and prepare for database loading
    │
    ▼
    PostgreSQL (pgAdmin)
    • Design relational database schema
    • Create tables with primary and foreign keys
    • Load data from Python into PostgreSQL
    • Validate record counts and confirm successful data load
    │
    ▼
    SQL Analysis
    • Data quality checks
    • Exploratory data analysis
    • Create reporting views
    • Aggregate business metrics for dashboard reporting
    │
    ▼
    Python (Jupyter Notebook)
    • Query reporting views
    • Export curated datasets to CSV
    │
    ▼
    Tableau
    • Connect to curated CSV files
    • Build interactive dashboards and visualizations
    • Create executive-ready reports and data stories
    │
    ▼
    GitHub
    • Publish SQL scripts
    • Share Python notebooks
    • Document project methodology
    • Showcase Tableau dashboard and business insights
